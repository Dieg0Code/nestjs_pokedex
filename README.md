# Ejecutar en modo desarrollo

1. Clonar repositorio
2. Ejecutar

```
npm install
```

3. Se necesita tener Nestjs instalado

```
npm i -g @nest/cli
```

4. Ejecutar base de datos

```
docker-compose up -d
```

## Stack

- MongoDB
- Nest
- Typescript
- Docker
